- DMA漏失控制的核心功能，主要包括两个方面：
    - 一个是对各个独立计量区域的漏失水平进行评估和比较，有效确定漏失水平最严重的区域，提高管网暗漏检测的针对性
    - 二是根据夜间最小流量数据的变化，当管网漏失增长时及时预警，实现管网漏点的快速发现、定位和修复
